# DevHire - Modern Job Portal

DevHire is a comprehensive, modern job portal designed to streamline the hiring process for both recruiters and candidates. Built with a focus on user experience and efficiency, it provides a robust platform for job posting, application management, and career growth.

## 🚀 Features

### For Candidates
- **Smart Job Search**: Filter and find jobs that match your skills.
- **Personalized Dashboard**: Track applications and manage your profile.
- **Help Guides**: Detailed resources to help you navigate the platform.
- **Profile Management**: Keep your professional details up to date.

### For Recruiters
- **Job Posting**: Easily create and manage job listings.
- **Applicant Tracking**: View and manage candidate applications through a dedicated dashboard.
- **Recruiter Help Center**: Resources for efficient hiring.

## 🛠️ Tech Stack

- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI, Lucide Icons, Framer Motion (Animations)
- **Form Management**: React Hook Form, Zod (Validation)
- **Routing**: React Router DOM
- **Data Visualization**: Recharts

## 🏗️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AKSH-NAIK/Devhire-UI.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
